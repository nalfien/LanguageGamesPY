# LanguageGamesPY

A recreation of my language games app in Python.
Currently has:
-Palindromes: Takes the plain-text sentence, and copies it in reverse. e.g., "taco" becomes "tacocat"
-Encrypt:
--Letter-by-letter, looks at whether the letter is a consonant or vowel.
--Based on the letter's position in the string, shifts that by the position in whichever array.
--e.g. "aaaaa" becomes "eioua"
